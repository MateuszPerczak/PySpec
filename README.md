# PySpec

# PySpec is a system profiling and monitoring application for Microsoft Windows that detects the cpu, gpu, ram, motherboard and other devices.

## Python version:

3.8.2

## Used Libraries:

- tkinter
- wmi
- os
- json
- dataclasses
- GPUtil
- typing
- platform
- traceback
